# 🏙️ Mumbai Rent Intelligence Platform

> An enterprise-grade Data Analytics project that transforms raw Mumbai rental listings into actionable business intelligence using PostgreSQL and Power BI.

---

# 📌 Project Overview

The **Mumbai Rent Intelligence Platform (MRIP)** is an end-to-end Business Intelligence project designed to analyze Mumbai's residential rental market.

The project simulates a real-world analytics solution for property portals, real estate consultants, investors, and tenants by converting raw rental listing data into meaningful insights.

This project demonstrates the complete analytics lifecycle:

- Business Understanding
- Data Engineering
- Data Quality Assessment
- Advanced SQL Analytics
- Data Warehouse Design
- Power BI Dashboard Development
- Business Storytelling
- Documentation
- Version Control using Git & GitHub

---

# 🎯 Business Problem

Finding the right rental property in Mumbai is difficult because pricing varies significantly across localities, property sizes, furnishing types, and market demand.

There is no simple way to answer questions such as:

- Which locality offers the best value for money?
- Which areas are overpriced?
- How much rent should a tenant expect?
- Which locations provide the highest rental yield?
- How is the rental market changing over time?

This platform answers those questions using data.
